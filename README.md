## 概述
一个基于monorepo框架的微前端系统，底层使用pnpm + wujie实现，用于收录前端应用实例。

## 启动
```bash
npm i pnpm -g
pnpm i
# 各应用会以.env文件中配置的端口名进行启动
# 在mac中该命令不可用，需分别启动主、子应用
pnpm -F * dev
```
